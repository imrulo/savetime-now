# SaveTime.Now - Professional Web Design Service

A modern, high-converting website for SaveTime.Now, a done-for-you web design service targeting busy entrepreneurs, coaches, and consultants.

## Features

- **Next.js 14+** with App Router for optimal performance
- **TypeScript** for type safety
- **Tailwind CSS** for responsive design
- **SEO Optimized** with proper meta tags, Open Graph, and JSON-LD schema
- **Mobile-First** responsive design
- **Accessibility** compliant (WCAG 2.1 AA)
- **Core Web Vitals** optimized

## Tech Stack

- Next.js 15.5.6
- React 18
- TypeScript
- Tailwind CSS
- Next.js Font Optimization

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

### Vercel (Recommended)

1. Push your code to GitHub
2. Connect your repository to Vercel
3. Deploy automatically

### Manual Deployment

1. Build the project:
```bash
npm run build
```

2. Start the production server:
```bash
npm start
```
